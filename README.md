武汉大学移动端Totem数据库系统

## 任务要求

### Task 1: 实现对象Union操作

### Task 2: 实现更新迁移操作

### Task 3: 实现手机端各app产生的轨迹（百度地图、跑步软件、共享单车、打车等），并进行Union操作以完整的存储个人出行轨迹

所有功能需要用SQL实现进行调用，可利用javacc进行编译，小组成员共四人，能连通3大任务，并进行流畅的展示
